A Google Colab-based learning project developed as part of the STKI (Sistem Teknologi / Sistem Terkait Informasi) practical assignment under the guidance of Lecturer: Pasnur.

This project implements Cosine Similarity-based Information Retrieval to measure similarity between data entries and validates results using an Excel output file.

---

## 🚀 Project Description

This project demonstrates how Similarity Retrieval (Cosine Similarity) works in an Information System context using Python in Google Colab.

The system performs:
- Data processing and preparation
- Vector-based similarity computation
- Ranking of similarity results
- Exporting results to Excel
- Comparison between Colab output and Excel file
